Project Description

Developed a custom drone platform with a tailored flight controller for autonomous and manual flight operations. Integrated sensors (IMU, GPS, barometer) with real-time control algorithms for stable navigation and obstacle avoidance. Implemented communication protocols for remote operation and payload control. Delivered a modular system supporting FPV racing configurations, smart payload deployment, and integration with ROS for advanced autonomy testing.

  • Drone system integration (motors, ESCs, sensors)
  • PID & sensor fusion algorithms (IMU, GPS, barometer)
  • FPV racing setup and payload control
  • ROS / ROS2 integration for autonomy
  • Wireless communication (RC, telemetry, Wi-Fi)
