Project Information

  • Project Title: Autonomous UGV with Robotic Arm
  • Your Role: Embedded Systems & Robotics Engineer
  • Category: Robotics & Autonomous Systems
  • Platforms: ROS/ROS2, MCU + SBC, MoveIt
  • Project Type: Custom Development

Project Description

Designed and developed an autonomous unmanned ground vehicle (UGV) equipped with a robotic arm for pick-and-place operations. Implemented SLAM-based navigation for autonomous mapping and obstacle avoidance. Integrated multi-sensor fusion (LiDAR, camera, IMU) with ROS for real-time decision-making and path planning. Programmed the robotic arm for precise manipulation, enabling the UGV to autonomously detect, grasp, and place objects in dynamic environments.

Skills & Deliverables

  • ROS / ROS2 navigation stack & SLAM
  • Robotic arm control (MoveIt, kinematics, motion planning)
  • Multi-sensor fusion (LiDAR, camera, IMU)
  • Autonomous path planning & obstacle avoidance
  • Embedded control integration (MCU + SBC)
  • Pick-and-place automation
